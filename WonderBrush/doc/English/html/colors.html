Colors

There's a special area reserved for colors in Wonderbrush's user interface. A "currently set color" (the bigger panel to the left), that applies to new objects, as well as a number of smaller panels which can be saved and loaded as a palette from the Color menu. The currently set color can be put into one of the smaller color panels via drag and drop. In many other areas of the program drag&dropping of colors is also supported (for example with the Object list to set the color of the selected objects). Clicking the currently set color opens a more detailed color editor that allows a finer tuning of a color than it is already possible with the two color editing panels below the palette.

If the gradient tool is selected and gradient editing is active, the current color is automatically transferred to the active gradient point for easier adjustment.
