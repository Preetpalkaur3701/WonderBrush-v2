Objects

Abstractly spoken, objects fill a layer with contents. For that, most objects create a kind of mask or stencil that is applied in a certain Mode to the already existing objects. A mask is simply an opacity channel, that defines for every pixel of the object how strongly it affects the opacity of that pixel.

There are different types of objects, that differ mainly in how they produce their mask. Most of them offer a number of Properties that affect the mask creation.

To create an object of the desired type, the corresponding tool has to be activated. Every tool offers controls that can be manipulated to influence the properties of the object to be created. To actually make the object, you simply click into the canvas and, if needed, drag with mouse or pen. Some tools, especially the shape tool, offer various editing possibilities while creating the object. Filter objects are a special case, as they can only be produced by choosing them from the Filter menu. There's no tool for bitmap objects either, since they represent imported pictures or were generated by invoking Object->All->Freeze. Generally, new objects appear on top in the active layer (i.e. the last entry in the list). They can be moved to another position in the list by a simple drag&drop. Objects can be duplicated by holding down the hift key while doing this. Some types of objects, like the shape and the text tool, can be edited again with the respective tool by double clicking their entry in the Object list. Other object types can only be edited through their Property list.
